# Airflow DAG: Загрузка курсов валют ЦБ РФ

## Описание
DAG для ежедневной загрузки курсов валют с сайта Центробанка России.

## Особенности
- Парсинг XML данных с cbr.ru
- Загрузка в ClickHouse
- Обработка ошибок
- Логирование
- Для анализа доступна [тестовая выгрузка курсов валют](docs/выгрузка_из_бд.csv) за период с 19.06.2025 по 28.06.2025.
